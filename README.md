# 🐱 VidaPet Felina

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um website estático para uma clínica veterinária especializada em gatos.

O objetivo foi aplicar conceitos fundamentais de HTML e CSS, incluindo estruturação de páginas, organização visual, navegação e responsividade básica.

---

## 🎯 Objetivo
Desenvolver um website utilizando apenas HTML e CSS, aplicando os fundamentos de:

- Estruturação de páginas
- Organização visual
- Navegação entre páginas
- Semântica HTML
- Responsividade básica

---

## 📝 Descrição
O site apresenta uma clínica veterinária fictícia chamada VidaPet Felina, com foco exclusivo no atendimento de gatos.

O projeto possui múltiplas páginas interligadas, contendo:

- Página inicial (Início)
- Página institucional (Sobre)
- Página de contato
- Página informativa (FIV & FeLV)

---

## 🧩 Funcionalidades

- Menu de navegação funcional  
- Identidade visual moderna  
- Conteúdo organizado  
- Imagens ilustrativas  
- Formulário de contato  
- Tabela de horários  
- Página educativa sobre doenças felinas  
- Responsividade básica

## ✅ Requisitos Atendidos

- Projeto desenvolvido individualmente  
- Utilização apenas de HTML e CSS  
- Mínimo de 3 páginas interligadas  
- Navegação funcional entre páginas  
- Uso de arquivo CSS externo  
- Estrutura semântica com header, nav, main, section, footer  
- Uso de títulos, parágrafos, listas, links, imagens, formulário e tabela  
- Identidade visual coerente  
- Organização visual e espaçamento  
- Responsividade básica  
- Sem uso de frameworks
